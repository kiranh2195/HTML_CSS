Selectors in CSS - > it is in key:value pair 
================
syntax:
selector{
	key:value;
	key:value;
}

shortcut for getting suggestion of tags ===>>   < + CTRL +SPACE 

Types 
1)tag selector -> h1{}
2)id selector -> #id{}
3)class selector -> .class{}

tag selector 
example->
        h1{
            color: black;
        }
id selector 
example->
<head>
    <title>CSS Basics</title>
    <Style>
        #html{
            color: blueviolet;
        }
        #css{
            color: red;
        }
        #js{
            color: yellow;
        }
    </Style>
</head>
<body>
    <h1 id="html">Welcome to HTML</h1>
    <h1 id="css">Welcome to CSS</h1>
    <h1 id="js">Welcome to Java Script</h1>
</body>

class selector->
<head>
    <title>CSS Basics</title>
    <Style>
        .red{
            color: red;
        }
        #js{
            color: yellow;
        }
    </Style>
</head>
<body>
    <h1 class="red">Welcome to HTML</h1>
    <h1 class="red">Welcome to CSS</h1>
    <h1 id="js">Welcome to Java Script</h1>
</body>
